Если вы здесь то помните: 
Этот каталог служит для описания самой платформы на которой разворачиваем систему Эльфов, а это значит:
Подкаталоги: 
 "include/" 	- подключаемые хедеры необходимые для сборки всех проектов.
 "libs/" 	- актуальные библиотеки необходимые для сборки практически всех проектов.
 "loaders/"	- актуальные стабильный и тестовый загрузчик (обработчик формата).
 "ToolChain/"	- инструменты для сборки исходников и работы с платформой.
 "Doc/"		- инструкции по настройке утилит и примеры пользования средой.
 "elfs/"	- распределенную структурированную коллекцию приложений собраных под платформу 
			при помощи объявленного инструментария.
И, ПОМНИТЕ - это еще только часть работы следующая - правильное наполнение окружения под нужды пользователя :)
